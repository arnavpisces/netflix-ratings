# Netflix Ratings Extension

Chrome extension that shows Rotten Tomatoes and IMDb ratings on Netflix titles, plus lets you blacklist shows (like Cocomelon) to mark them as "Low Quality."

## Installation

1. Clone this repo
2. Open Chrome → `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" → select the extension folder
5. Done

## Usage

- Ratings appear automatically on Netflix titles
- Click the extension icon to add shows to your blacklist
- Blacklisted shows get a red overlay and "Low Quality" badge
